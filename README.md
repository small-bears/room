# room
基于springboot vue自习室预约系统
## 一、 项目介绍
(协同过滤推荐算法+AI智能助手)基于SpringBoot的自习室管理系统包含前台用户模块、后台设备管理员模块、后台管理员模块三部分，具体功能如下：

前台用户模块基础功能模块：首页、自习室、商品信息、设备信息、公告信息、购物车、智能 AI个人中心功能：修改密码、自习室预约、取消预约、签到信息、签退信息、失物招领、设备报修、维修分配、维修信息、我的订单、我的收藏
后台设备管理员模块管理操作类功能：系统首页、设备信息管理、维修分配管理、维修信息管理、保养计划管理、保养信息管理、设备巡检管理、设备报废管理、个人中心
后台管理员模块全流程管理类功能：系统首页、设备管理员管理、用户管理、区域管理、自习室管理、自习室预约管理、取消预约管理、签到信息管理、签退信息管理、失物招领管理、失物标记管理、商品类型管理、商品信息管理、设备类型管理、设备信息管理、设备报修管理、维修分配管理、维修信息管理、保养计划管理、保养信息管理、设备巡检管理、设备报废管理、系统管理、订单管理、个人中心
